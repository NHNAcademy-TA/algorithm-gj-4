# 소수

### [ 문제 ]

- 주어진 범위 사이의 소수 개수를 구합니다.

### [ 조건 ]

1. 먼저 1부터 n 사이의 소수 개수를 구해봅니다.
   - `isPrime()` 구현
2. 1번을 완료한 후, m부터 n 사이의 소수 개수를 구해봅니다.
    - `isPrimeSolution()` 구현

### [ 예시 ]

|   입력    | -> | 출력  |
|:-------:|----|:---:|
|  1, 10  |    |  4  |
|  1, 50  |    | 15  |
| 1, 1000 |    | 168 |
|  5, 20  |    |  6  |
