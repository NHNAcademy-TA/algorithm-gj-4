# 문자열 속 특정 문자열 갯수 세기

### [ 문제 ]

- 문자열 속에 특정 문자열이 몇 번 등장하는지 개수를 세어봅니다.

### [ 조건 ]

- `contains()` `subString()` 사용 금지 

### [ 예시 ]

|        입력         | -> | 출력 |
|:-----------------:|----|:--:|
|  abciwkabc, abc   |    | 2  |
|   aciwcosck, c    |    | 3  |
|    banana, na     |    | 2  |
| alhorithmtest, th |    | 1  |
