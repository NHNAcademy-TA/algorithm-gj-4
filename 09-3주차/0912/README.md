# 최대 최대공약수

### [ 문제 ]

- 주어진 정수 배열로 만들 수 있는 가장 큰 최대공약수를 구합니다.
- 예를 들어, 주어진 정수 배열이 { 10, 20, 30, 40 } 인 경우,
    - { 10, 20 }, { 10, 30 }, { 10, 40 }, { 20, 30 }, { 20, 40 }, { 30, 40 } 로 조합할 수 있고,
    - 각각 10, 10, 10, 10, 20, 10 의 최대공약수를 가집니다. 이 중 가장 큰 최대공약수는 20 입니다.

### [ 조건 ]

1. 없습니다.

### [ 예시 ]

|       입력       | -> | 출력 |
|:--------------:|----|:--:|
| 10, 20, 30, 40 |    | 20 |
|    5, 7, 12    |    | 1  |
