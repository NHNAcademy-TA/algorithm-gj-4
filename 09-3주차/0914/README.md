### [ 문제 ]

- 소수를 분수로 변경하기

1. 소수점 자리 수를 계산 `getDecimalNumber()`을 구현
2. Fraction의 구현
3. solution 구현

### [ 조건 ]

- 입력 값은 -1 < num < 1 사이 숫자만 생각합니다.
- 무한 소수는 생각하지 않습니다. (나눠 떨어지는 수만 생각)
- BigDecimal을 사용합니다.
- Type 변환은 `(int)`로 변환합니다.

### [ 예시 ]

| 입력   | -> | 출력   |
|------|----|------|
| 0.1  |    | 1/10 |
| 0.25 |    | 1/4  |
